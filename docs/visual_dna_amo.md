# Visual DNA - AMO ISSO

## Fonte-Mae (obrigatoria)
- Referencia principal: `prints/AMO ISSO.jpg`
- Todo novo componente deve preservar:
  - textura de papel
  - tinta preta dominante
  - destaque ocre/amarelo como acento editorial
  - tipografia com contraste editorial (titulo) + operacional (interface)

## Sistema de Tema
- Tema padrao: `xilo`
- Tema ambar: `xilo-amber`
- Tema invertido: `xilo-invert`
- Nao usar outros temas na experiencia principal.

## Regras de Composicao
- Titulo principal: peso alto, serif editorial, espaco amplo.
- Texto de apoio: sans tecnico limpo.
- Secoes: ritmo vertical constante; evitar blocos "apertados".
- Destaques: usar marca-texto (acento), nao gradientes decorativos.

## Iconografia
- Icones podem manter desenho original.
- Devem receber textura/ranhura visual via filtro de tinta.

## Motion
- Transicoes de estado devem usar "tinta":
  - abrir pomodoro
  - travar sessao
  - alternar ambiente/tema
- Duracao curta, entrada organica, saida limpa.

## Nao Pode
- Neon/brilho digital como linguagem principal.
- Glassmorphism predominante.
- Multiplicidade de paletas concorrentes.
- Tipografia sem hierarquia editorial.
